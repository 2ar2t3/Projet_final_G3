affichage_streamlit module#

Streamlit dashboard ― ETS_en_Turbulence#

Interface graphique du projet MGA802 (ÉTS Montréal).

Fonctionnalité#

  1. Instancie une seule fois le collecteur temps réel main.Main (thread daemon) et le stocke dans :pydata:`st.session_state`.

  2. Récupère à chaque rafraîchissement la copie thread-safe app.to_display (NumPy (N, 4) : lon, lat, alt, diam).

  3. Affiche la carte des turbulences via affiche_carte.Data + affiche_carte.Carte.

  4. Montre une légende (couleur, taille, opacité) et un exemple de cisaillement sous forme de bulles bleues.

  5. Auto-rafraîchit la page toutes les 3 s grâce à streamlit_autorefresh.st_autorefresh().
